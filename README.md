# Test Practice Others Workspace

This repository contains a diverse set of folders and files for practicing programming, data science, algorithms, and machine learning across multiple languages and domains. Below is an overview of the main contents and their purposes:

## Folder Structure & Contents

### Dart/
- Dart language practice scripts, including basic algorithms and exercises.

### DSC 10/
- Assignments 1-5: R scripts and CSVs for data science coursework.
- Project Test: R project files for plotting and analysis.

### DSC 11/
- Text files and C programs for algorithms (Knapsack, Backtracking, Dynamic Programming, Graphs, etc.).
- PDF for assignment mapping.

### DSC 11 Practicals/
- C++ source and executable files for classic algorithms: sorting, graph, tree, and dynamic programming.
- Practicals.txt: Summary and notes for practicals.

### DSC 12/
- DBMS practicals, MySQL queries, and relational algebra notes.

### DSC 15/
- Case studies and notes for advanced coursework.

### DSE DS/
- Assignments 1-4: Jupyter notebooks, CSVs, and PDFs for regression, clustering, and classification.
- DS Project: Keras models and stock dataset for deep learning.
- Neural Networks: Python scripts and notebooks for neural network experiments.

### js_express/
- Node.js/Express server code and frontend for JS experiments.

### practice/
- HTML, JS, and image files for web development practice.

### Python/
- CN Programs: Python scripts for computer networks (client/server, CRC, Dijkstra, Stop-and-Wait, etc.).
- GE CBMR Practical: Sentiment analysis results and notebooks for practicals.
- LPP Project: Linear programming project files, flight scheduling datasets, and notebooks.
- Miscellaneous Python scripts and notebooks.

## Notable Files
- README.md: This file.
- .venv/, .vscode/: Environment and editor settings.
- Porsche_Sentiment_Results.xlsx, Practical_Results.xlsx: Output files from sentiment analysis.

## How to Use
- Browse folders for code samples, assignments, and data science projects.
- Run Jupyter notebooks for interactive analysis (see DSE DS, Python/LPP Project, Neural Networks).
- Use Python scripts for algorithm and ML practice.
- Explore C++ and C programs for classic algorithms in DSC 11 Practicals.
- Web and JS experiments are in practice/ and js_express/.

## Requirements
- Python 3.x, Jupyter, pandas, matplotlib, scikit-learn, keras, textblob (for ML/data science folders).
- C++/C compiler for algorithm practicals.
- Node.js for JS/Express experiments.
